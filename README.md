Real-time multiplayer mini games (actual game in French)

Current status

## Installation

```
cd client
npm i
cd ../server
vim .env # with variables PORT and PORT_SSL
npm i --save-dev
npm start
```

In another tab

```
cd client
npm run dev
```

## Docs

Generate docs

```
cd server
npm run docs
```
